# 题目源文件规范

ExamGen 使用 Markdown 编写试卷，用专用标记区分不同类型的内容区块，以便解析引擎准确识别和渲染。

## 文件结构

```
---
YAML 元数据
---

题目正文
```

## 一、YAML 元数据

| 字段 | 类型 | 必填 | 取值 | 说明 |
|------|------|------|------|------|
| `title` | string | **是** | — | 试卷标题 |
| `subject` | string | 否 | — | 科目 |
| `time` | int | 否 | — | 考试时长（分钟），触发倒计时 |
| `total_score` | float | 否 | — | 试卷总分（仅展示） |
| `default_score` | float | 否 | `1.0` | 未指定分值时默认分 |
| `shuffle` | bool | 否 | `false` | 打乱题目顺序 |
| `option_shuffle` | bool | 否 | `false` | 打乱选项顺序 |
| `passing_score` | float | 否 | — | 及格分 |

## 二、题目内容区块

每道题由若干**区块**组成，使用带 `#` 前缀的专用标记声明区块类型。题目之间用空行分隔。

### 区块标记一览

| 标记 | 含义 | 可选/必填 | 说明 |
|------|------|-----------|------|
| `#题干` | 题目主干 | **必填** | 描述题目内容，支持 Markdown |
| `#图片` | 插图/配图 | 可选 | 一张或多张图片 |
| `#表格` | 数据表格 | 可选 | 题干中需要展示的表格 |
| `#选项` | 选择题选项 | 必填(选择/判断题) | 四行 `- A. xxx` 格式 |
| `#答案` | 参考答案 | **必填** | 判分依据 |
| `#分值` | 分值 | 可选 | 不填则用 `default_score` |
| `#解析` | 题目解析 | 可选 | 提交后展示 |

### 区块通用规则

- 每个标记独占一行，以 `#` 开头
- 标记后的所有内容直到下一个 `#` 标记或题目结束都属于该区块
- 区块内支持完整 Markdown 语法（表格/代码/列表/图片等）

### 题型标识

每道题第一行为 `#题号. [题型]`，题型取值：

| 标识 | 题型 | 含义 |
|------|------|------|
| `#1. [单选]` | 单选题 | 四选一 |
| `#2. [多选]` | 多选题 | 四选多 |
| `#3. [判断]` | 判断题 | 对/错 |
| `#4. [填空]` | 填空题 | 手动输入 |
| `#5. [简答]` | 简答题 | 文本作答 |

## 三、题型详细格式

### 单选题 `[单选]`

```markdown
#1. [单选]
#题干
下列哪个是 Python 的关键字？

#选项
- A. class
- B. Class
- C. CLASS
- D. classify

#答案
A

#分值
2

#解析
`class` 是 Python 保留关键字，用于定义类。
```

### 带图片和表格的单选题

```markdown
#2. [单选]
#题干
观察下面的流程图和参数表，选择正确的描述：

#图片
![流程图](https://example.com/diagram.png)

#表格
| 参数 | 值 | 说明 |
|------|----|------|
| time | 90 | 考试时长 |
| score | 100 | 总分 |

#选项
- A. 网络处于用户均衡状态
- B. 网络处于系统最优状态
- C. 网络未收敛
- D. 以上都不对

#答案
A

#分值
3

#解析
根据 Wardrop 第一原则，UE 状态下被使用路径行驶时间相等且最小。
```

### 多选题 `[多选]`

```markdown
#3. [多选]
#题干
以下哪些是 Python 内建数据类型？

#选项
- A. list
- B. dict
- C. array
- D. tuple

#答案
A,B,D

#分值
3

#解析
`array` 不是内建类型，需要 `import array`。
```

### 判断题 `[判断]`

```markdown
#4. [判断]
#题干
Python 中 `None` 表示空值。

#答案
A

#分值
2
```

> 判断题为 `A=正确`、`B=错误`。可省略 `#选项`，引擎自动补全。

### 填空题 `[填空]`

```markdown
#5. [填空]
#题干
Python 中输出到控制台的函数是 `____`，不换行输出用 `____`。

#答案
print|end

#分值
2

#解析
`print()` 默认以换行符结尾，`print(..., end='')` 可取消换行。
```

> 多个空位用 `____` 表示，答案用 `|` 分隔。

### 简答题 `[简答]`

```markdown
#6. [简答]
#题干
请简述 Python 中列表和元组的主要区别。

#答案
列表与元组的核心区别：
- **可变性**：列表可变，元组不可变
- **语法**：列表 `[]`，元组 `()`
- **用途**：元组可作为字典键

#分值
5

#解析
可变性是最核心的区别。元组创建后元素不可增删改。
```

> 简答题提交后仅展示参考答案，不做自动判分（标记为"待评"）。

## 四、富文本支持

### 题干/解析/答案均支持完整 Markdown

```markdown
#题干
请阅读以下**重点内容**并回答问题：

1. 第一点
2. 第二点
   - 子项 A
   - 子项 B

> 提示：注意区分概念。

`inline code` 示例。

#解析
| 对比项 | 方案A | 方案B |
|--------|-------|-------|
| 复杂度 | O(n)  | O(n²) |
| 内存   | 低    | 高    |

- [x] 已确认项
- [ ] 待确认项
```

### 数学公式

使用 `$...$`（行内）或 `$$...$$`（块级），在线环境由 KaTeX 渲染，离线时降级为纯文本：

```markdown
#题干
函数 $f(x) = \sum_{i=1}^{n} x_i^2$ 在 $x=0$ 处取得最小值。

#解析
$$
\frac{\partial f}{\partial x_i} = 2x_i = 0 \implies x_i = 0
$$
```

### 图片

支持远程 URL 和本地相对路径（生成的 HTML 会保留原始引用）：

```markdown
#图片
![网络拓扑](https://example.com/topo.png)
### 或本地文件
![交通图](./assets/map.png)
```
