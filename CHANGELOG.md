## 2026-06-24

### 提交 & 打印体验

- **打印成绩单重设计** — 新增独立打印页，`Ctrl+P` 只输出成绩单（不打印全部题目）。布局：标题 + 科目 → 总分大字 → 是否及格 → 各题型正确数 → 逐题 #N 题干 ✓/✗ N分
- **未答题提示整合到侧栏** — 提交时未答题以红色标签显示在导航栏进度条下方，点击可跳转，不再弹窗
- **导航栏折叠** — 侧栏标题右侧 `◀` 按钮，点击折叠/展开，主内容区自动扩展

### 判分

- **分区内乱序** — `shuffle` 改为在每个 `## 分区` 内独立打乱，保持分区之间的顺序（单选 → 多选 → 判断 → 填空 → 简答）
- **简答关键词半自动判分** — 解析中 `关键词：A, B, C` 行会被提取，按匹配比例自动给建议分，人工在批阅面板中可调整
- **答案放在解析之前** — 提交后正确答案显示在解析上方，阅读顺序更合理
- **答案公式渲染** — 答案中的 LaTeX 公式（如 `$R \geq 0.8$`）正确渲染，不再被 `escapeHTML` 破坏
- **填空多答案容错** — 单个空多个可接受答案（`|` 分隔），答对一个即判对

### 答题体验

- **localStorage 答案保存** — 刷新/误关页面不丢答案，包括书签状态
- **键盘快捷键** — `1-4` 选 ABCD / `Tab` 跳题 / `Enter` 提交
- **题目标记（书签）** — 侧边栏 `☆` + 题卡分值旁 `☆`，点击变 `★`，保留到成绩复盘
- **导航栏结果汇总** — 提交后在侧边栏顶部显示总分（渐变紫底白字）+ 各题型正确数，逐题 ✓/✗/?

### 批阅

- **批阅翻题按钮** — 右面板增 `◀ 上一题 / 下一题 ▶` 按钮
- **分值按钮即点即选** — 点击分值按钮直接选中亮高，无需手动输入

### 文档 & 规范

- `CHANGELOG.md` — 新建本文件
- `docs/spec.md` — 补全 YAML 常见错误表、填空多答案规则、简答关键词说明
- `docs/prompt.md` — 新建出卷提示词模板，含格式规范、关键词规则
- `tests/fixtures/sample.md` — 扩展铁律 10→12 条、添加关键词示例
- `README.md` — 重写功能特性列表（16 项），更新项目结构

### 错误反馈

- **ParseError 结构化异常** — `course`→`subject`、`score`→`total_score`、`time: 90分钟`→`time: 90` 等常见错误会精确定位字段并给出修复建议
- **Web 端 JSON 错误返回** — `/generate` 接口异常时返回 `{error, message, field, suggestion}`，前端分三行展示

## 2026-06-23

### 答题体验

- **公式渲染修复** — `mdToHTML` 先保护 `$...$` / `$$...$$` 占位 → marked 渲染 → 还原公式，避免 `_` `*` 被 marked 解析破坏
- **填空题公式保护** — `buildFillInput` 中 `**` 占位符替换时不破坏公式内的指数符号
- **答案直观展示** — 每道题卡片底部显示绿色 `✓ 回答正确` 或红色 `✗ 正确答案：X`，不再需要悬停
- **导航栏对错状态** — 提交后在左侧栏逐题显示绿色 `✓` / 红色 `✗` / 紫色 `?`

### 解析器

- **_strip_leading_backslash 智能检测** — 仅剥离 AI 生成级转义的 `\[选项\]`，不破坏 LaTeX 公式 `\[\frac{a}{b}\]`
- **YAML 字段名校验** — `course`、`score` 等无效字段预检报错

### 文档

- `docs/spec.md` — 正式版本，含铁律 11 条、题型详细格式、富文本支持、常见错误排查
- `tests/fixtures/sample.md` — 标准示例模板，含注释说明
